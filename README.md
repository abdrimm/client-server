# client-server
Simple low-level client-server connection.

* Download repository:
```bash
git clone https://github.com/abdrimm/client-server
```
*Compile
```bash
make
```

*Run server
Argument: port
```bash
bin/server 500
```

*Run client
Argument: ip and port
```bash
bin/client 127.0.0.1 5000
```
